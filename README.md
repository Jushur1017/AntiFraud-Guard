# 🛡️ AntiFraud Guard (反诈卫士)

> 专为老年人设计的“防屏幕共享诈骗”安全辅助工具。
> **实时监控 + 强力阻断 + 蜂鸣报警，守护家人的钱袋子。**

---

## 📖 项目背景 (Why this?)

近年来，针对老年人的电信诈骗层出不穷，其中最猖獗的手段之一就是诱导老人开启**“屏幕共享”**功能，远程查看验证码并转走资金。

此 App 旨在解决这一痛点：**当手机处于“屏幕共享”状态，且用户打开“银行/支付类软件”时，本应用会立即触发最高级别的声光报警，并强制覆盖屏幕，物理阻断转账操作。**

## ✨ 核心功能 (Features)

* **🕵️ 实时侦测：** 智能识别后台运行的会议软件（如腾讯会议、Zoom 等）。
* **🔒 双重验证：** 仅在“屏幕共享”+“打开银行App”同时发生时触发，不影响日常使用。
* **🚫 强力阻断：** 弹出无法关闭的全屏红色警告窗，遮挡支付界面。
* **📢 蜂鸣报警：** 调用系统底层警报音（不受静音模式影响），发出刺耳警报震慑骗子，警醒老人。
* **✅ 纯净安全：** 无需联网权限，不上传任何用户隐私数据，代码完全开源。

## 📱 支持的检测范围

**会议软件：** 腾讯会议、Zoom、钉钉会议、TeamViewer、向日葵远程控制等。
**敏感应用：** 支付宝、微信支付、各大银行 App。

## 🛠️ 使用说明 (How to use)

1.  **安装：** 下载 APK 并安装到手机（安装时请信任“未知来源”）。
2.  **授权（核心步骤）：** App 启动后，请务必开启以下三个权限：
    * **无障碍服务 (Accessibility)：** 用于检测当前打开的应用。
    * **通知读取权限：** 用于检测后台是否有会议软件运行。
    * **悬浮窗权限：** 用于弹出红色警告遮挡层。
3.  **锁定后台：** 建议在多任务界面将本 App **锁定/加锁**，并允许其自启动，防止被系统清理。
